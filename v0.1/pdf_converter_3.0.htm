Version Anglaise.

Configuration requise :

  • Ordinateur équipé d'un processeur Intel® Pentium® ou supérieur
  • Microsoft Windows® 98 (à partir de la version SE), Windows Millenium Édition, Windows 2000 ou Windows XP
  • Microsoft Internet Explorer 5.1 ou supérieur
  • Microsoft Word 97, Microsoft Word 2000, Microsoft Word XP ou Microsoft Word 2003
  • 64 Mo de RAM (128 Mo recommandés)
  • 90 Mo d'espace disque pour les fichiers de l'application, plus 5 Mo pour installation
  • 5 Mo pour Microsoft Installer (MSI) si ce programme n'est pas déjà installé. (Ce programme est déjà installé si vous utilisez Windows Millenium Édition, Windows 2000 et Windows XP.)
  • Moniteur SVGA en mode 256 couleurs et offrant une résolution d'affichage de 800 x 600 pixels ; affichage en mode Couleurs 16 bits recommandé (option appelée " 65536 couleurs " sous Windows 2000 et
  • Moyenne (16 bits) " sous Windows XP)
  • Dispositif de pointage compatible Windows
  • Lecteur de CD-ROM pour l'installation
  • Connexion à Internet pour l'activation et la mise à jour du logiciel.

"ScanSoft PDF Converter Professional 3 est la solution PDF complète à l'usage des professionnels. PDF Converter Professional offre des fonctions inégalées en matière de création, de conversion et d'édition de fichiers PDF, ainsi que l'ajout de notes de révision et la possibilité de sécuriser vos fichiers PDF. Avec PDF Converter Professional, les utilisateurs bénéficient d'un contrôle total de leurs documents et gagnent en productivité. Cette application révolutionnaire vous permet notamment d'apporter des modifications directement dans des fichiers PDF, aussi simplement que si vous travailliez sur les fichiers d'origine. Elle permet également de convertir instantanément des formulaires PDF Statique en formulaires PDF prêts à être remplis électroniquement. PDF Converter Professional propose également de nouveaux outils d'annotation et d'ajout de commentaires, ainsi que de puissantes fonctions de sécurité."
